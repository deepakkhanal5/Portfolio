# Teacher Portfolio Website

A professional, responsive portfolio website for educators. This portfolio showcases teaching experience, educational background, skills, and achievements.

## 📋 About

This is a **static portfolio website** created for a **teacher at Ishwori Secondary School** who is currently pursuing a Bachelor's degree while working professionally as an educator.

### Key Information
- **Status**: Teacher + Bachelor Student
- **Institution**: Ishwori Secondary School
- **Location**: Nepal
- **Focus**: Education, Student Development, Teaching Excellence

## 🌟 Features

### Responsive Design
- Fully responsive and mobile-friendly
- Works seamlessly on desktop, tablet, and mobile devices
- Hamburger menu for mobile navigation

### Sections Included

1. **Navigation Bar** - Sticky navigation with smooth scrolling
2. **Hero Section** - Eye-catching introduction
3. **About** - Personal introduction and teaching philosophy
4. **Experience** - Teaching experience and responsibilities at Ishwori Secondary School
5. **Education** - Academic background and continuous learning
6. **Skills** - Teaching skills, subject expertise, soft skills, and technical skills
7. **Achievements** - Certifications and accomplishments
8. **Contact** - Contact information and message form
9. **Footer** - Copyright and tagline

## 💻 How to Use

### Opening the Website
1. Open `index.html` in a web browser
2. Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

### Customizing Your Information
Edit `index.html` to add your personal information:
- Replace email address and phone number
- Update name and profile details
- Modify teaching subjects and experience
- Add specific achievements and skills

### Styling
Edit `style.css` to customize the design:
- Colors (defined in `:root` variables)
- Fonts and typography
- Spacing and layout
- Responsive breakpoints

### Functionality
Edit `style.css/script.js` to add features:
- Form submission handling
- Analytics integration
- Additional interactive elements
- API connections
